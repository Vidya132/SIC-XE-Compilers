Command to compile and run:(Windows)
	>>g++ -o assembler assembler.cpp
	>>assembler

Assumptions:
1. "input.txt" file with input assembly code should be in the same directory as the assembler.
2. Label,Opcaode,Operand each have exactly 10 characters.
3. Four files will be generated at the end of execution.
	i)	intermediate.txt
	ii)	listing.txt
	iii)	Symboltable.txt
	iv)	output.txt
4. Comments should have "." as their label.
5. If there is any error it will be printed on the terminal.
6. The hexadecimal constants of type X'' are assumed to be of even number of digits.
7. There are nospaces before and after ',' while using indirect addressing
